# وحدة صيانة المقار والمباني | Facilities Maintenance Module

نظام لتسجيل ومتابعة بلاغات صيانة مباني ومقار شركة **وطن الأول للموارد البشرية** (السكنات، الفروع، مراكز التوزيع، المكاتب) حتى الإغلاق، مع الصيانة الوقائية المجدولة.

## المحتويات
```
frontend/maintenance.html   ← النظام (يفتح مباشرة في المتصفح)
database/schema.sql         ← هيكل البيانات (مرجع للمرحلة الثانية)
docs/maintenance-SOP.md     ← الإجراء التشغيلي المعياري ومستوى الخدمة (SLA)
```

## كيف تجرّبه
افتح ملف `frontend/maintenance.html` بأي متصفح. يعمل بدون إنترنت (عدا الخط)، ويحفظ البيانات محلياً على نفس الجهاز.

## المميزات
- لوحة معلومات بمؤشرات فورية (مفتوح / قيد التنفيذ / عاجل / مغلق هذا الشهر).
- تسجيل بلاغ صيانة وتحويله عبر المراحل: مفتوح ← قيد التنفيذ ← مغلق.
- تصنيف حسب الموقع، التصنيف، الأولوية، مع بحث وفلترة.
- إدارة المواقع والمباني.
- الصيانة الوقائية المجدولة مع تنبيه عند الاستحقاق.

## المرحلة الثانية (لاحقاً)
ربط النظام بـ **Google Sheets** ليصير قاعدة بيانات مشتركة بين مشرفي المواقع.
